# Yemeksepeti Android Bootcamp Bitirme Projesi

Bitirme projesi olarak gerçekleştirdiğim proje yer almaktadır.

##Kullanılan Teknolojiler
 - MVVM architecture
 - Dagger Hilt
 - Retrofit
 - Coroutine
 - Glide
 - View Binding
 - Navigation
 - View Pager (Onboarding)
 - Recycler View
